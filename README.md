# Sauna Word Search

It's a program that takes a 2D array of strings as input, and calculates and draws the path and words through the array on a canvas.

## Eruption

> Boilerplate for React/Typescript, built on top of Vite ⚡️

_It's fast! Even the tests are fast, thanks to Vite with Vitest ⚡️_
